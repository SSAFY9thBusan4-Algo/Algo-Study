# 2166 G5 다각형의 면적
##### <p align="right"> 2023.05.28 📆 </p> 

 
### 👩‍🏫 풀이
1. 다각형의 면적은 삼각형으로 나눠서 구할 수 있다.
2. 다각형을 이루는 순서대로 주어지기 때문에 반복문을 통해 구했다.
3. 첫번째 점을 기준으로 2개씩 순서대로 고르면서 삼각형의 면적을 구했다.
4. 답이 음수가 나오는 경우가 있어 절댓값으로 출력했다.
